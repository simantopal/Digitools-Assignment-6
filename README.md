# DigiTools – Premium Digital Tools Web App

DigiTools is a modern web app that provides a curated collection of digital products, productivity tools, templates, and AI-powered assets for creators and professionals. Users can browse products, add them to the cart, and manage purchases seamlessly. The app is fully responsive, interactive, and built with modern React and Tailwind CSS technologies.

## ⚡ Main Technologies
- **React.js (ES6+)** – Component-based architecture and state management  
- **Tailwind CSS / DaisyUI** – Modern, responsive, and clean UI  
- **React Toastify** – Interactive notifications for actions like adding/removing products  
- **Font Awesome & React Icons** – Vector icons for UI elements  
- **Google Fonts** – Custom typography  

---

## ✨ Key Features
- **Interactive Product Listing:** Browse premium digital tools with descriptions, pricing, and tags.  
- **Add to Cart & Checkout:** Add products to the cart, view total price, remove items, and proceed to checkout.  
- **Dynamic Status Updates:** Cart and checkout actions update in real-time using React state.  
- **Dashboard & Stats:** View product highlights and usage statistics.  
- **Step-by-Step Onboarding:** Guides users to get started in 3 simple steps: Create Account → Choose Products → Start Creating.  
- **Pricing Plans:** Clear, transparent pricing with starter, pro, and premium plans.  
- **Responsive Design:** Works perfectly on mobile, tablet, and desktop screens.  
- **Interactive Footer & Navigation:** Accessible navigation, social media links, and footer links for resources and company info.
